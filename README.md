## 📖 About The Project

KUTA Collections is a modern e-commerce platform dedicated to Ethiopian traditional clothing (Habesha Kemis), featuring bilingual support (English/Amharic), AI chatbot, and 50+ custom animations.

### ✨ Key Features

- 🌐 **Bilingual Support** - English & Amharic with language switcher
- 🤖 **AI Chatbot** - Smart FAQ and customer support
- 🎨 **50+ Animations** - Smooth, engaging interactions  
- 📱 **Fully Responsive** - Works on all devices
- 🛍️ **Dynamic Filtering** - Search and filter products
- 🗺️ **Google Maps** - Store location integration
- 🎭 **3D Product Cards** - Interactive flip animations

## 🛠️ Built With

- **React.js** - Frontend framework
- **Context API** - State management
- **CSS3** - Custom animations
- **Google Maps API** - Location services
